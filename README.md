### üöÄ Featured Project: 2Do Enterprise Task Management

A production-grade task management ecosystem built with a focus on cloud-native architecture and automated deployment.

* **Backend Architecture**: Developed with **Java 21** & **Spring Boot 3.4**, utilizing a **stateless RESTful design** for maximum scalability.
* **Advanced Security (Redis)**: 
    * **JWT Blacklisting**: Implemented secure logout functionality by blacklisting tokens in **Redis**.
    * **API Rate Limiting**: Integrated **Redis-based rate limiting** to prevent API abuse and ensure system resilience.
* **Performance & Caching**: Leveraged **Azure Cache for Redis** to reduce database load and provide sub-millisecond data retrieval.
* **Frontend Architecture**: Modular **Angular 17** using Standalone Components and reactive state management (**RxJS**).
* **Cloud & DevOps**: 
    * Orchestrated with **Docker** and hosted on **Azure Container Apps** & **Static Web Apps**.
    * Fully automated **CI/CD** pipelines via **GitHub Actions** for seamless deployments.
